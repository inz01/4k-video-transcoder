🔄 How to Regenerate This Report

Run the KPI comparison tool from the project root:

source .venv/bin/activate

# Cloud URL is auto-detected from OpenStack if credentials are sourced.
# Or set it explicitly with --cloud-url:
python kpi_compare.py \
  --local-url http://127.0.0.1:8000 \
  --run-tests \
  --video videos/sample_video_1093662-hd_1920_1080_30fps.mp4 \
  --presets 480p,720p,1080p \
  --output kpi_report

# To specify cloud URL manually:
# python kpi_compare.py --cloud-url http://<FLOATING_IP>:8000 --run-tests ...
  • Charts are saved to kpi_report/
  • Open kpi_report/index.html in your browser to view this dashboard
  • Use --no-charts for terminal-only output
  • Use --local-file / --cloud-file for offline JSONL input
